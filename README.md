# churn_attrition
Churn Attrition is a combined analysis examining customer and employee loss in a business using Kaggle's Telco Churn and HR Employee Attrition Data Sets.

## Customer Churn
Customer Churn measures how many customers have left the telco and for what reasons.

## Employee Attrition
Employee Attrition measures how many employees have left the enterprise and for what reasons.
